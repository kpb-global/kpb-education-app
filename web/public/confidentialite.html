KPB Education

Politique de Confidentialité

Dernière mise à jour : Avril 2026

1. Responsable du traitement

KPB Education (ci-après « nous », « notre », « KPB ») est responsable du traitement des données personnelles collectées via l'application mobile KPB Education.

Contact : contact@kpbeducation.com

2. Données collectées

Nous collectons les données suivantes :

  • Identité : nom complet, adresse e-mail, numéro de téléphone, WhatsApp
  • Profil académique : niveau d'études, niveau cible, compétences linguistiques, notes, filières d'intérêt, pays de destination
  • Données d'utilisation : résultats d'orientation, éléments sauvegardés, historique de recherche
  • Données techniques : identifiant de l'appareil (pour les notifications push), données analytiques anonymisées via Firebase Analytics
  • Diagnostics : rapports d'incidents et erreurs techniques via Firebase Crashlytics ; métriques agrégées de synchronisation des contenus (succès ou échec, durée, source locale/cache)

3. Finalités du traitement

Vos données sont utilisées pour :

  • Personnaliser vos recommandations de filières, pays et bourses
  • Gérer vos dossiers de candidature et vos rendez-vous
  • Vous envoyer des notifications pertinentes (mises à jour de dossier, deadlines)
  • Améliorer nos services via des statistiques agrégées et anonymisées
  • Communiquer avec vous dans le cadre de vos démarches

4. Base légale

  • Consentement : vous acceptez cette politique lors de la création de votre compte
  • Exécution du contrat : le traitement est nécessaire pour fournir nos services
  • Intérêt légitime : amélioration de nos services et sécurité de la plateforme

5. Partage des données

Vos données ne sont PAS vendues à des tiers.

Elles peuvent être partagées avec :

  • Nos conseillers internes pour le suivi de vos dossiers
  • Nos partenaires institutionnels (universités) uniquement avec votre accord explicite lors de la soumission d'un dossier
  • Firebase / Google (Analytics, Crashlytics, Cloud Messaging) pour l'analytique, la stabilité et les notifications push
  • Les autorités compétentes si la loi l'exige

6. Durée de conservation

  • Données de profil : conservées tant que votre compte est actif
  • Données de dossier : conservées 3 ans après la clôture du dossier
  • Données analytiques : agrégées et anonymisées, conservées indéfiniment
  • En cas de suppression de compte : vos données personnelles sont supprimées sous 30 jours

7. Sécurité

  • Communication chiffrée via HTTPS/TLS
  • Mots de passe hachés (bcrypt)
  • Tokens d'authentification sécurisés (JWT)
  • Données sensibles (tokens) stockées dans le Keychain (iOS) / Keystore (Android)
  • Accès restreint aux données de production

8. Vos droits

Conformément au RGPD et aux lois applicables, vous avez le droit :

  • D'accéder à vos données personnelles
  • De rectifier vos données
  • De supprimer votre compte et vos données (voir la procédure de suppression)
  • De limiter le traitement
  • De vous opposer au traitement
  • De portabilité de vos données

Pour exercer ces droits, contactez-nous à : privacy@kpbeducation.com

9. Firebase Analytics

Nous utilisons Firebase Analytics (Google) pour collecter des données d'usage anonymisées :

  • Écrans consultés, actions effectuées (orientation, recherche, sauvegarde)
  • Aucune donnée personnelle identifiable n'est transmise à Google
  • Vous pouvez désactiver la collecte depuis les paramètres de votre profil

10. Modifications

Cette politique peut être mise à jour. Nous vous informerons de tout changement significatif via une notification dans l'application.

En continuant à utiliser l'application après une mise à jour, vous acceptez la nouvelle politique.
