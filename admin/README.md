# KPB Education Admin

Next.js admin scaffold for KPB Education counselors, commercial staff, and admins.

## Target responsibilities

- request and case assignment
- counselor follow-up
- content operations
- role attribution
- operational monitoring

## Current state

This workspace currently provides the first route structure and dashboard shell. It still needs:

- authentication
- real data fetching
- mutation forms
- analytics charts
- content editing workflows
